香港BGP V3的测评数据也不错，电信、联通、移动的延迟基本都在30-50ms之间，这个延迟水平在香港VPS里算是中上水平了。

### 稳定性反馈

有用户连续监控了三个月，除了机房计划内维护，基本没遇到过宕机。工单响应速度也挺快，一般几个小时就能得到回复，英文、中文都能沟通。

当然也有人吐槽，说偶尔会有丢包的情况，特别是晚高峰。不过这个问题在共享带宽的VPS上很难完全避免，毕竟价格摆在那儿，不能要求太多。

### 性能跑分

有技术宅做了详细的性能测试，UnixBench跑分987.8分（1核1GB配置），磁盘读写平均591MB/s，这个成绩在同价位产品里属于中等偏上。

虚拟化用的是KVM，资源隔离做得还行，不会出现某个邻居疯狂占用资源导致你的VPS卡成翔的情况。
